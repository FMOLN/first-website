# FMOLN Perfumes 🖤✨

A premium fragrance e-commerce storefront built for **FMOLN Perfumes**, based in Addis Ababa, Ethiopia.

Customers can browse 10 luxury perfumes, view product details, add to cart, and place orders directly via WhatsApp — no payment gateway needed.

## 🛍️ Features

- Browse 10 authentic luxury perfumes
- Product detail pages with fragrance profiles
- Add to cart with quantity tracking
- Order via WhatsApp with pre-filled message

## 🚀 Deployment

This site is deployed on **Netlify** with **Netlify CMS** for product management.

### Deploy your own copy:
1. Fork this repository
2. Log in to [netlify.com](https://netlify.com)
3. Click **Add new site → Import from GitHub**
4. Select this repository
5. Click **Deploy** — no build settings needed
6. Enable **Netlify Identity** and **Git Gateway** in site settings

---

## 🛒 How Ordering Works

1. Customer browses the shop and adds perfumes to cart
2. Fills in name, phone, and delivery area
3. Clicks **Place Order** — opens WhatsApp with a pre-filled order message
4. FMOLN confirms and arranges **Pay on Delivery**

## 📦 Tech Stack

- Pure HTML, CSS, JavaScript — no frameworks
- Netlify CMS for product management
- localStorage for cart persistence
- WhatsApp Business API for orders
